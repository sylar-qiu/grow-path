# 芯片设计：从设计到量产全流程

芯片（IC）从设计到量产的全流程、关键环节、核心术语及风险点梳理，用于内部参考或与客户沟通时快速查阅。

---

## 一、核心流程与关键环节（按时间线）

### 第一阶段：设计准备与启动

**核心动作：** 客户（Fabless 设计公司）定义芯片规格，启动设计。

**关键产出：** 电路设计图、技术方案。

**核心环节与关键词：**

- **工艺选择** — 确定制造工艺节点（如 55nm, 40nm, 28nm）
- **PDK 申请** — 向晶圆厂申请"工艺设计工具包"，这是设计的"工具箱"
- **建 Code** — 在晶圆厂开通客户账户，获得下单资格。高门槛（如 SMIC 要求月产能承诺）

**风险与价值点：**

| | 内容 |
|---|---|
| 风险 | 新建 Code 门槛高，周期长 |
| 价值 | 通过既有渠道（如已有渠道资源）帮助客户绕过建 Code 门槛，快速启动 |

---

### 第二阶段：物理设计与验证（流片前核心）

**核心动作：** 将电路图转化为物理版图，并反复验证。

**关键产出：** GDSII 文件（芯片制造的最终图纸）。

**核心环节与关键词：**

- **后端设计** — 布局布线，生成 GDS
- **DRC / LVS 验证：**
  - **DRC** — 设计规则检查，确保版图可制造
  - **Waive** — 对某些 DRC 错误的"豁免"决策
  - **Retooling** — 因修改设计而需重新制作光罩，导致成本增加与延期
- **Tape-Out（流片）** — 正式将 GDS 提交给晶圆厂

**风险与价值点：**

| | 内容 |
|---|---|
| 风险 | DRC 问题反复、最后一刻修改引发 Retooling |
| 价值 | 协助高效沟通，争取免费修改机会，控制 NRE 成本 |

---

### 第三阶段：制造准备与光罩制作

**核心动作：** 晶圆厂处理数据，制作生产模板。

**关键产出：** 光罩。

**核心环节与关键词：**

- **JDV** — 晶圆厂的"数据打包与最终确认"。这是修改的最后免费关口，JDV 释放后修改代价巨大
- **Mask Making** — 制作光罩
- **NRE** — 一次性工程费用，主要指光罩费。计算公式：NRE ≈ 单层光罩价 × 总层数 + 工程费

**风险与价值点：**

| | 内容 |
|---|---|
| 风险 | 错过 JDV 确认截止期，或 Retooling 层数失控 |
| 价值 | 推动客户完成 JDV 确认，锁定修改范围，避免项目"下车" |

---

### 第四阶段：晶圆制造、封装与测试

**核心动作：** 用光罩生产晶圆，切割封装成芯片。

**关键产出：** Wafer → Chip。

**核心环节与关键词：**

- **流片** — 在晶圆厂进行制造
- **CP / FT 测试** — 晶圆探针测试、成品芯片测试
- **封装** — 将裸片封装为成品芯片
- **Wafer Cost** — 每片晶圆的制造费，决定量产成本
- **MPW / Full Mask：**
  - **MPW** — 多项目晶圆，拼单降低 NRE，用于小批量
  - **Full Mask (NTO)** — 全掩模，用于大规模量产

**风险与价值点：**

| | 内容 |
|---|---|
| 风险 | 良率问题、产能排期紧张 |
| 价值 | 提供有竞争力的 Wafer Cost、保障产能、缩短流片周期 |

## 二、核心关键词 / 术语汇总

### 参与方

| 关键词 | 简要解释 |
|---|---|
| Fabless | 无晶圆厂设计公司（如你的客户） |
| Foundry / 晶圆厂 | 芯片制造厂（如 SMIC） |
| IDM | 集成器件制造商（设计制造一体） |
| Service Provider | 服务商 / 代理 |

### 成本与模式

| 关键词 | 简要解释 |
|---|---|
| NRE | 一次性工程费用（主要是光罩费） |
| Wafer Cost | 每片晶圆的制造费（量产主要成本） |
| MPW | 多项目晶圆，拼单流片 |
| Full Mask / NTO | 全掩模 / 新产品量产流片 |
| Retooling | 重新制作光罩（代价高昂） |

### 设计文件

| 关键词 | 简要解释 |
|---|---|
| PDK | 工艺设计工具包 |
| GDS / GDSII | 芯片最终版图文件 |

### 制造流程

| 关键词 | 简要解释 |
|---|---|
| Tape-Out | 提交 GDS，流片 |
| JDV | 数据最终确认环节（关键节点） |
| Mask / 光罩 | 生产用的模板 |
| Wafer / 晶圆 | 上有数百颗芯片的硅片 |
| Die / 裸片 | 晶圆切割后的单个芯片 |
| Chip / 芯片 | 封装后的成品 |

### 验证

| 关键词 | 简要解释 |
|---|---|
| DRC | 设计规则检查 |
| LVS | 版图与电路图一致性检查 |
| Waive | 对 DRC 错误的豁免决定 |

### 商务

| 关键词 | 简要解释 |
|---|---|
| Code | 客户在晶圆厂的账户代码 |
| Shuttle / 班次 | MPW 的轮次计划 |
| 良率 | 合格芯片的百分比 |

## 三、商务沟通核心要点（价值主张）

在与设计公司老板沟通时，可将以上知识转化为你的价值：

1. **"我们帮您解决准入和成本"** — 通过我们的渠道，您可以绕过 SMIC 严苛的建码门槛，并享受基于我们大客户资源的优势量产单价。

2. **"我们帮您管控流程风险"** — 在流片前的关键阶段（DRC 验证、JDV 确认），我们能协助高效沟通，争取免费修改、控制 Retooling 范围，保护您的 NRE 预算和上市时间。

3. **"我们提供产能与周期保障"** — 我们能提供确定的产能和更短的流片周期，助您产品快速上市。
