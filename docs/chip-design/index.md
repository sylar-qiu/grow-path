## 核心流程速览

```
设计准备 → 物理验证 → 制造准备 → 晶圆制造 → 封测 → 量产
                        ↓
                     JDV 确认
                （最重要决策节点）
```

- **PDK** = 开工许可证
- **Tape-Out** = 设计结束、制造开始
- **JDV** = 最后免费修改窗口
- **NRE** = 一次性成本（光罩费）
- **Wafer Cost** = 量产核心成本
- **Retooling** = 成本超支头号杀手
