# 第四章：关键术语深度辨析

这一章专门拆解你在实际邮件和对话中遇到的术语。每一个词我都按"它是什么 → 它在业务里代表什么 → 它对你跟客户沟通意味着什么"三层来写。

---

## PDK vs IP

这是最容易混淆，也最能体现你跟客户专业深度的一对概念。

### PDK（工艺设计工具包）

**它是什么：**
- 晶圆厂提供给设计者的"设计规则和基础元器件库"
- 包含：DRC/LVS 规则文件、晶体管/电阻/电容等基础器件的模型和符号、EDA 工具工艺库文件
- 每个晶圆厂、每个工艺节点独立一套

**它在业务中代表什么：**
- PDK 是"开工许可证"。没有 PDK，设计公司什么也做不了
- 申请 PDK 需要：资质审核 + 签署 NDA + 技术评估（2-4 周）
- PDK 通常免费，但绑定必须在该晶圆厂流片
- PDK 产权 100% 属于晶圆厂——你不能把 SMIC 的 PDK 拿去给华虹用

**跟客户沟通时：**
- 客户说"我拿到了 SMIC 的 PDK" → 意味着他已经通过了最基础的审核门禁，不再是"来路不明的小公司"
- 客户说"PDK 还没拿到，在等" → 这就是你的切入点。你能帮客户加速拿 PDK

### IP（知识产权核）

**它是什么：**
- 基于 PDK 设计好的、可复用的复杂功能模块
- 举例：ARM CPU 核、USB 控制器、DSP、DDR 控制器
- 不是基础元器件，而是"预制好了的子系统"

**它在业务中代表什么：**
- IP 是昂贵的"高级预制件"——跟 PDK 的免费不同
- 授权模式：
  - **一次性授权费**（几万到几百万美金不等）
  - **版税（Royalty）**——每卖一颗芯片抽 1-5%
- IP 供应商是 ARM、Cadence、Synopsys 这样的公司，不是晶圆厂
- IP 选择直接决定了芯片的性能和成本，是设计阶段最核心的决策之一

**跟客户沟通时：**
- 客户说"这个芯片用的是 ARM M4 核" → 意味着他已经花了 IP 的钱，设计已经做了一半
- 客户说"还没选好 IP" → 这意味着他还在设计早期
- IP 授权的谈判周期比流片更长（3-6 个月），如果你的客户在等 IP，流片时间会跟着推迟

### 核心区别表

| | PDK | IP |
|---|---|---|
| 比喻 | **砖瓦水泥和建筑规范** | **预制好的厨房或楼梯** |
| 提供方 | 晶圆厂 | 第三方 IP 公司 |
| 价格 | 通常免费 | 高额授权费 + 版税 |
| 获取难度 | 资质审核 + NDA | 复杂商业谈判 |
| 使用限制 | 绑定该晶圆厂 | 可使用于多家 |
| 开放程度 | 晶圆厂核心机密 | 公开市场购买 |

**实际业务场景中两者的关系：**

PDK 和 IP 不是二选一，而是**先后关系**。
1. 客户先拿到 PDK（晶圆厂准入）
2. 在 PDK 基础上选择并购买 IP（ARM、Cadence 等）
3. 基于 PDK + IP 做完整设计
4. 生成 GDS → 流片

## Die vs Chip

### 区别

| | Die（裸片） | Chip（芯片成品） |
|---|---|---|
| 状态 | 从晶圆切下来的单个芯片，未经封装 | 经过封装 + 测试的最终产品 |
| 对外形 | 裸露的硅片，极小、极脆 | 黑色/塑封的芯片，可以焊在 PCB 上 |
| 测试 | 仅 CP 测试（晶圆级测试） | CP + FT（封装后完整测试） |
| 成本 | 基准价 | Die 价 + 封装费 + 测试费（约增 20-30%） |
| 使用 | 需要自己封装才能用 | 直接可以用 |

### 行业惯例

**为什么 MPW 默认交付 Die：**
- 客户拿 Die 去自己评估（做可靠性测试、送样给终端客户）
- 客户可能有自己的封装合作方
- 跳过封装测试降低本次流片的直接费用

**什么情况下交付 Chip：**
- 量产项目，直接用于终端产品
- Turn-Key 项目（交钥匙）
- 客户没封装测试资源，你一并做了

### 跟客户沟通时

这是报价时最容易出歧义的地方。一定要明确问："你要 Die 还是 Chip？"

```
错误示例：
客户问："流片多少钱？"
你答："50 颗 1.8 万美金"
结果客户以为 1.8 万拿到的是一颗颗可以直接用的成品芯片
实际拿到的是裸露的 Die
客户不满 → 信任受损 → 后续难做

正确示例：
客户问："流片多少钱？"
你答："Die 是 1.8 万；如果要封装成成品 Chip，加封装和测试
      大概再加 4000-6000。您这边是自己封装还是需要我们一并？"
→ 客户心里有数，不会后期产生预期差
```

## 工艺类型辨析

你会在邮件中遇到客户说"我要 0.18 BCDE"或"0.18 RF"或"40nm LL"。同一个工艺节点（如 0.18μm）下有多种变体，应对不同应用。

### BCDE（集成模拟+数字+高压+存储）

**它是什么：**
- 0.18μm 工艺下的一种"特色工艺套件"
- B = Bipolar（双极型，用于模拟信号）
- C = CMOS（互补金属氧化物半导体，用于数字电路）
- D = DMOS（双扩散金属氧化物半导体，用于高压功率器件）
- E = eNVM / EPROM（嵌入式非易失性存储器）

**应用场景：** 电源管理芯片、显示驱动芯片、LED 驱动芯片
**特征：** 需要同时处理模拟信号、数字控制和高压输出

**跟客户沟通时：**
- 客户说"0.18 BCDE" → 大概率是做电源管理或显示驱动的
- BCDE 的工艺比纯 CMOS 贵，因为光罩层数更多（集成更多功能）
- 单一芯片实现所有功能 = 板级体积更小、成本更低

### RF（射频工艺）

**它是什么：**
- 在基础 CMOS 工艺上优化，专门处理高频无线信号
- 提供高性能无源器件（电感、电容）、低噪声晶体管、精确射频模型

**应用场景：** 蓝牙、Wi-Fi、手机射频前端、NFC
**特征：** 必须处理 GHz 级别的高频信号，对噪声和功耗要求极高

**跟客户沟通时：**
- 客户问"0.18 RF PDK" → 在设计蓝牙/Wi-Fi 类芯片
- RF 工艺比同节点数字工艺贵一档，但比 BCDE 便宜
- RF 设计难度大，也是 Retooling 高发区（高频信号对版图极其敏感）

### 同级别的其他工艺类型

| 工艺类型 | 关键词 | 应用场景 |
|---|---|---|
| **混合信号** | Mixed-Signal | 模拟 + 数字结合，但不含高压器件 |
| **高压** | HV（High Voltage） | 显示驱动、电源管理（几十-上百伏） |
| **嵌入式存储器** | eFlash / eEEPROM | MCU、带存储功能的芯片 |
| **低功耗** | LL（Low Leakage / Low Power） | 物联网、可穿戴设备 |
| **标准逻辑** | Logic / Digital | 纯数字芯片，最基础、最便宜 |

**行业惯例：**
- 同一个工艺节点下，特色工艺（BCDE、RF、HV）的光罩层数比纯数字逻辑多 30-50%
- 特色工艺的 PDK 更新慢于标准工艺——如果你想等某个特色工艺的新版 PDK，可能需要等半年
- 客户有时会"选错"工艺——比如需要模拟功能但选了纯数字工艺。这是前期技术沟通不到位导致的

## FSR（Foundry Service Request）

### 它是什么

"流片服务请求表"——客户在正式流片前向晶圆厂提交的工艺选项和制造规范表格。

### 它在业务中代表什么

FSR 是**制造合同的技术部分**，是 JDV 生成的依据。

**FSR 填什么：**
- 工艺节点（如 0.18 BCDE）
- 工作电压（如 pure5V / 3.3V / 1.8V）
- 器件类型
- 金属层数
- 特殊模块（eFlash、MIM 电容等）
- 测试结构
- 可靠性要求

**为什么 FSR 极其重要（你在邮件中已经体会到了）：**
- FSR 上选"pure5V" vs "3.3V" → 决定了某些光罩层（NW/PW 层）要不要做
- FSR 填错 → JDV 错 → 光罩错 → 芯片功能不对 → 整批报废
- FSR 通常由 Design Service 的人帮客户填，但**最终客户签字**

### 行业惯例

- 许多项目在 FSR 阶段反复确认（你提到的邮件里双方确认"pure5V"就是这个环节）
- FSR 确认后，任何改动都涉及费时费钱的修改流程——你之前需要知道这个
- 如果客户让你参与技术讨论，你应该帮客户多一重检查，确认 FSR 上的参数跟设计意图一致

## 其他业务术语速查

| 术语 | 它是什么 | 业务场景中的含义 |
|---|---|---|
| **Tape-Out** | 将最终 GDS 提交给晶圆厂的行为 | 设计阶段结束，制造阶段开始。既是里程碑也是担责时刻 |
| **JDV** | GDS 数据的最终确认 | 最后免费修改窗口，过时不候 |
| **Mask / 光罩** | 生产用的光学模板 | NRE 的大头成本来源 |
| **Wafer / 晶圆** | 上有数百颗芯片的硅片 | 基本计价单位 |
| **良率** | 合格芯片百分比 | 直接影响有效成本，是 Foundry 的技术水平体现 |
| **NRE** | 一次性工程费 | 主要是光罩费。沉没成本，但决定项目门槛 |
| **Waive** | DRC 错误的豁免 | 晶圆厂工程师的"批准权"，有限度的柔性处理 |
| **Retooling** | 重新做光罩 | 银子打水漂的代名词 |
| **Code** | 客户在晶圆厂的账户代码 | 生意的准入证 |
| **Shuttle / 班次** | MPW 的周期性运行计划 | 错过一次等几个月 |
| **Turn-Key** | 交钥匙方案 | 从设计到封装测试全包，省心但贵 |
